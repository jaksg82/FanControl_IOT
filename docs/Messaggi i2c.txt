Request messages:

GETSTS		Get status
GETRNG      Get temperature range

SETT##      Set temperature range values
				byte[4] -> TargetTempMin
				byte[5] -> TargetTempMax


Response messages:

STS######	STS     -> Header
			byte[3] -> Sensor 0 Temperature
			byte[4] -> Sensor 0 Humidity
			byte[5] -> Sensor 1 Temperature
			byte[6] -> Sensor 1 Humidity
			byte[7] -> Fan PWM Percentage
			byte[8] -> Fan Relay Status: 0=Off 1=On

RNG##		RNG     -> Header
			byte[3] -> Temperature Range Min
			byte[4] -> Temperature Range Max
